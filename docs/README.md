# 文档目录（docs）

本目录存放项目的所有文档，包括需求文档、接口文档以及后续的技术文档。

## 📄 核心文档

### 需求文档

- **`简易需求文档.md`** ⭐ **核心文档**
  - 项目核心需求说明
  - 前后端功能划分
  - 模型与参数约定
  - **这是项目实现的主要参考文档**

- **`需求文档V1.md`**
  - 详细的需求文档（V1.0 版本）
  - 包含完整的功能需求、参数说明、接口草案等
  - 适合深入了解系统设计细节

### 接口文档

- **`接口文档.yaml`**
  - OpenAPI 3.0 格式的接口定义
  - 包含所有 API 端点的请求/响应 Schema
  - 可用于生成 API 文档或客户端代码

## 🔍 文档使用建议

1. **开始开发前**：先阅读 `简易需求文档.md` 了解核心需求
2. **接口开发**：参考 `接口文档.yaml` 了解 API 规范
3. **深入了解**：查阅 `需求文档V1.md` 获取详细设计信息
