- 实现一个时变海浪环境模型，实现对海浪高度的模拟和查询
    - 前后端分离逻辑，减少耦合
    - 后端核心逻辑
        - 区域模拟渲染，给定一个区域，对区域内进行海浪模拟，输出区域海浪高度
        - 单点查询，给定一个点，使用区域模拟结果，进行插值，输出该点海浪高度。采用空间双线性插值
        - 输入区域（矩形区域，使用经度纬度（对角两个点描述），深度），结合风向、地形、波浪谱进行模拟

        - 初始化初始状态，表示初始风场和初始海浪海浪状态，海浪初始状态可以随机，但要符合规则。
        - 第0刻表示初始状态，步长暂定为200ms（可配置）
        - 间隔步长进行区域模拟，实现一个同步时钟，便于与前端对应，
    - 前端显示逻辑
        - 区域模拟渲染，可视化区域，
        - 单点查询，配合鼠标点击，或者输入坐标，进行查询，返回海浪高度
        - 输入参数配置，给定默认参数，用于海浪模型的模拟，支持前端修改，并根据修改的参数进行区域模拟
        - 使用fastapi构造web服务
        - 前端提供start按钮，点击开始则开始海浪模拟，前端设定渲染的步长，与后端计算步长不同，是前端的渲染步长，防止卡顿
        - 可配置的参数都显示到前端中，方便调试

    - 后续约定输入、输出的具体格式
        - 区域采用对角两个点描述，使用经度、纬度、深度描述，深度用于控制海浪的高度，防止浅水区域过高的海浪
        - 输出为时刻、使用经度、纬度、海浪高度
        - 区域输出为区域范围 + 时刻 + （经度、维度、海浪高度）集
        - 配置参数，离散化间隔、离散化点上限
        - 配置参数，波浪谱模型选择，（先默认其中一个，后续可选择）
    - 海浪模型是由风场模型到波浪谱模型
        - 风场模型与波浪谱模型解耦
        - 风场模型后续改为变化风场模型
        - 波浪谱模型由瞬时风场模型生成
        - 谱到波浪的形成采用叠加法
    - 风场模型
        - 风场模型先为固定风模型，后续再使用其他风场模型
        - 参数：TODO
        - 参数可配置，显示到前端
    - 波浪谱模型
        - Pierson-Moskowitz 光谱或 JONSWAP 光谱，默认Pierson-Moskowitz后续再实现JONSWAP
        - 参数可配置，显示到前端
    - 数据格式：
        采用Json来管理
    - 输入输出为经纬度，中间处理可转换为xy网格
