## 前端（frontend）说明

本目录用于实现海浪环境模型的可视化前端，核心功能包括：

- 区域模拟渲染：展示给定区域的海浪高度场随时间变化
- 单点查询：通过鼠标点击或输入经纬度坐标，查询该点的海浪高度
- 参数配置：展示和调整风场、波浪谱模型及离散化参数等配置

### 技术栈规划

- 推荐使用：Vite + React + TypeScript
- 依赖管理：统一使用 `pnpm`（请避免使用 npm / yarn）

### 目录结构约定（当前仅占位）

- `public/`：静态资源
- `src/`
  - `components/`：通用基础组件
  - `pages/`：页面级组件
  - `services/`：与后端交互的 API 封装
  - `store/`：状态管理（若使用 Redux/Zustand/其他方案）
  - `styles/`：样式文件
  - `utils/`：前端通用工具

后续可以在 `frontend` 目录中通过类似命令初始化项目（示例）：

```bash
pnpm create vite .
```

当前阶段仅约定目录与说明，不生成具体前端代码。
